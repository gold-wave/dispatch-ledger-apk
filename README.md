## App Summary

Phone Dispatch is a mobile app for store-side used-phone dispatch ledger work. It helps stores without computers record devices, scan or recognize IMEI, create dispatch batches, export a standard Excel file, and share it through WhatsApp, Feishu, email, or other mobile channels.

门店可以直接在手机上完成以旧换新旧机寄送台账录入，包含店铺信息、发件人、设备 IMEI、机型、价格、寄送方式和运单号。App 会导出统一格式 Excel，方便后台统一收集、合并、查重和核查。
